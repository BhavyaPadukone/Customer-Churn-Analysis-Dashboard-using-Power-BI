# Telecom Churn Analysis : Customer Risk Analysis

## Overview

This project is a Power BI dashboard developed to analyze customer churn in the telecom industry.
The dashboard helps identify customer behavior, churn patterns, service usage, payment preferences, and retention risks using interactive visualizations.

The project uses:

* Power BI
* Power Query
* DAX
* Interactive Dashboard Design

---

## Objectives

* Analyze customer churn behavior
* Identify high-risk customers
* Study service usage patterns
* Understand customer demographics
* Support business retention strategies

---

## Features

* Customer Profile Analysis
* Churner Profile Analysis
* Interactive KPI Cards
* Tenure Analysis
* Internet Service Analysis
* Contract Type Analysis
* Payment Method Insights
* Q&A Dashboard for custom queries
* Interactive Navigation Buttons

## Key Insights

* Customers using Fiber Optic services show higher churn rates.
* Month-to-month contract users are more likely to churn.
* Electronic check payment users contribute significantly to churn.
* Customers with shorter tenure show higher churn probability.
* Streaming services and tech support influence customer retention.
